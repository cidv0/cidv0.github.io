Why install FiveM? ↓

FiveM is a multiplayer modification framework for GTA V that lets you play on customized dedicated servers run by communities.

  • Play roleplay, drifting, racing, deathmatch, or entirely custom game modes hosted by server operators.
  • Servers can add custom cars, maps, weapons, and scripts for experiences you cannot get in standard GTA V.
  • FiveM does not modify your GTA V installation, so you can switch between GTA Online and FiveM.

Where FiveM installs

The installer uses the folder you run it from if it is empty. Otherwise it defaults to:

%localappdata%\FiveM

If FiveM asks for your GTA V location, point it to the folder that contains:

GTA5.exe

Typical Steam path: ...\Steam\steamapps\common\Grand Theft Auto V\

Step-by-step

Steam GTA V + FiveM install flow

  1. Step 1

    Install and update GTA V in Steam

    Install GTA V through Steam, then launch it once so Rockstar and Social Club setup finishes and the game creates its first files. Make sure it is fully updated before moving on.

  2. Step 2

    Download the official FiveM client

    Grab FiveM.exe from the official site so you get the latest build and security checks.

  3. Step 3

    Run the installer

    Run FiveM.exe. Use an empty folder if you want a custom install location; otherwise it installs under %localappdata%\FiveM.

  4. Step 4

    Launch FiveM

    Start FiveM from the Start Menu or desktop. It will validate your GTA V ownership and may ask you to sign in to Rockstar if automatic validation fails.

  5. Step 5

    Select your GTA V folder if prompted

    Point FiveM to the folder that contains GTA5.exe. For most Steam installs it is: ...\Steam\steamapps\common\Grand Theft Auto V\.

Common gotchas

  • Run GTA V once from Steam before launching FiveM. This fixes most ownership and launcher detection issues.
  • FiveM uses the Steam listing Grand Theft Auto V Legacy (App ID 271590), not Grand Theft Auto V Enhanced. Stay on the default branch and keep it updated.
  • If FiveM fails to start, check antivirus or firewall interference and retry.
